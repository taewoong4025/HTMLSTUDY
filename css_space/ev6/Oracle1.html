What is Oracle?

Oracle의 특징

관계형 데이터베이스는 우리가 흔히 표현하는 행(Column), 열(Record)로 구성된Table간의 관계를 나타낼때 사용한다. 우리는 이렇게 표현된 데이터를 SQL(Structured Query Language)을 사용하여 데이터 관리 및 접근을 한다.

장점

  1. 범용적이고, 고성능이다.
  2. 데이터의 일관성 보증
  3. 정규화에 따른 갱신 비용 최소화

단점

  1. 대량의 데이터 입력 처리
  2. 갱신이 발생한 테이블의 인덱스 생성 및 스키마 변경
  3. 칼럼 확장의 어려움
  4. 단순히 빠른 결과
